# Mona Lisa Interativa - Projeto Alura

## 📋 Descrição
Este projeto cria uma versão interativa da Mona Lisa em Canvas, onde os olhos seguem o movimento do mouse em tempo real.

## 🎨 Características
- ✨ Desenho estilizado da Mona Lisa com as cores originais
- 👀 Olhos que seguem o cursor do mouse dinamicamente
- 🎭 Detalhes como cabelo, rosto, vestido e paisagem
- 📱 Responsivo para dispositivos móveis
- 🌈 Design moderno com gradientes e efeitos visuais

## 📁 Arquivos do Projeto

### `index.html`
- Estrutura principal da página
- Referencia CSS e JavaScript
- Canvas para desenho

### `styles.css`
- Estilo do container e página
- Gradiente de fundo bonito
- Efeitos de blur e sombra
- Responsividade

### `script.js`
- Lógica de desenho com Canvas
- Rastreamento do movimento do mouse
- Algoritmo para calcular direção dos olhos
- Uso de cores originais da Mona Lisa

## 🚀 Como Usar

1. Clone ou baixe os arquivos
2. Abra o arquivo `index.html` em um navegador
3. Mova o mouse pela tela e veja os olhos acompanharem!

## 🎨 Cores Utilizadas
- Pele: `#D4A574`
- Cabelo: `#3D2817`
- Olhos: `#6B4423`
- Vestido: `#2D5016`
- Fundo: `#8B7355`

## 💻 Compatibilidade
- Todos os navegadores modernos que suportam Canvas HTML5
- Funciona em desktop e dispositivos móveis (com toque)

## 📚 Conceitos Aprendidos
- HTML5 Canvas API
- Matemática (atan2, cos, sin) para movimento dos olhos
- Event listeners para mouse e touch
- Gradientes e formas em Canvas
- Curvas de Bezier para desenho suave
